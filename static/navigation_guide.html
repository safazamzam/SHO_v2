Navigation Guide

How to access the NEW SMTP UI Design

Step 1: Open the Secrets Dashboard

Navigate to the secrets management page:

http://127.0.0.1:5000/admin/secrets/

Make sure your Flask application is running first!

Step 2: Click the SMTP Tab (NEW DESIGN)

You'll see these tabs at the top of the page:

ServiceNow SMTP NEW DESIGN Application OAuth & SSO
Important: The SMTP tab should be active by default and shows a green "NEW DESIGN" badge. If you're seeing the old design, you might be on a different tab!
Step 3: What You Should See
NEW SMTP Design Features:
  • Modern card-based layout
  • Two-column responsive grid
  • Professional input styling
  • Enhanced typography and spacing
  • Contextual icons for each field
  • Better visual hierarchy
Still Seeing Old Design?
Check these things:
  • Make sure you're on the SMTP tab (not ServiceNow)
  • The SMTP tab should have a green "NEW DESIGN" badge
  • Try refreshing the page (Ctrl+F5)
  • Clear browser cache if needed
Still Having Issues?

If you're still seeing the old design:

  1. Take a screenshot of what you're seeing
  2. Check which tab is currently active (highlighted)
  3. Verify the URL in your browser
  4. Try opening the page in an incognito/private window
Note: Only the SMTP tab has been redesigned. The ServiceNow, Application, and OAuth tabs still have the old design.
