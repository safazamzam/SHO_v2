SSO Configuration Guide
Quick Setup Steps:
  1. Choose your identity provider type above
  2. Click "Configure" to set up provider details
  3. Test the configuration using the "Test" button
  4. Enable auto-provisioning if desired
  5. Users can then login via SSO
Important Notes:
  • SSO integrates with existing user roles
  • Auto-provisioning creates users automatically
  • Multi-tenant support preserves account isolation
  • Regular username/password login remains available
  • All SSO events are audited
